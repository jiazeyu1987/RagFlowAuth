OCR 准确率验证（4 图，单文件 HTML）

每张图：选择图片 → OCR（可选）→ 粘贴标准文本 → 计算准确率（CER / Accuracy）。
提示：你当前消息里只提供了 2 张图片路径，本页已预置在【图片 1】【图片 2】；请在【图片 3】【图片 4】选择剩余图片后再点“全部执行 OCR”。
说明 / 注意
1) OCR 使用 Tesseract.js（从 CDN 加载）。首次 OCR 会下载语言模型，可能较慢。
2) 若浏览器对本地文件/Worker 有限制，建议用本地 HTTP 服务打开：python -m http.server 8000，然后访问 http://localhost:8000/ocr_accuracy_eval.html。
3) 准确率定义：Accuracy = 1 - Levenshtein(识别, 标准) / 标准字符数；同时提供“去空白”口径用于弱化换行/空格差异。
