# 用户需求规格

版本: v1.0
更新时间: 2026-04-14

| URS ID | 对应条目 | 用户需求 |
|---|---|---|
| URS-007 | R7 | GMP 合规资料、版本、基线和周期复核状态必须保持一致，并可由仓库内门禁脚本复核。 |
| URS-011 | FDA-02 | 审计取证导出必须同时提供人可读副本与便携格式，并附带完整性摘要。 |
| URS-012 | FDA-03 | Controlled document register and review package export must be auditable and gate-validated. |
| URS-013 | GBZ-01 | 维护阶段的系统、数据库、接口、配置和预期用途变更必须经过影响判定，并在需要时触发再确认。 |
| URS-014 | GBZ-02 | 紧急变更必须先授权、后部署、再复盘，并保留授权依据、风险控制、复盘和 CAPA 证据。 |
| URS-015 | GBZ-03 | 已批准文档的退役、保留期访问和管理员取证导出必须走同一条受控链路。 |
| URS-016 | GBZ-04 | 供应商评估与环境确认资料必须形成受控文件，并保留 IQ/OQ/PQ 状态和仓库外残余项边界。 |
| URS-017 | GBZ-05 | 培训矩阵、培训确认记录与操作员资格状态必须可追溯，并可由仓库内门禁复核。 |
