# 角色与职责

版本: v1.0  
更新时间: 2026-04-02

| 角色 | 主要职责 |
|---|---|
| 系统管理员 | 管理用户、权限组、组织结构、数据安全配置；不得直接执行业务文件操作。 |
| 上传人员 | 按权限提交文档并跟踪审核结果。 |
| 审批人员 | 按流程执行审批/驳回/覆盖审批，并完成电子签名。 |
| 查看人员 | 在授权范围内查看文档、搜索结果和预览信息。 |
| QA/合规 | 复核验证资料、审计日志、恢复演练和发布证据。 |
| 运维 | 执行发布、回滚、备份、恢复、租户迁移和退役操作。 |

职责要求:

- 账号不得共用。
- 关键动作必须可归因到个人。
- 任何生产变更必须走变更控制和验证闭环。
