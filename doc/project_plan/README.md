# Project Plan Artifacts

This directory stores execution artifacts for the full delivery plan in `todolist.md`.

Files:
- `rtm.md`: Requirement Traceability Matrix (requirement -> module/API/UI/test)
- `dod_template.md`: Definition of Done checklist template
- `change_audit_template.md`: Change audit template for requirement/policy/permission/task changes

Usage:
1. Update `rtm.md` first when adding/changing requirement scope.
2. Create task tickets with `dod_template.md` checklist attached.
3. Record key changes with `change_audit_template.md` for governance and acceptance evidence.
