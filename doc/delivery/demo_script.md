# 验收演示脚本（RTM 对齐）

更新时间：2026-03-14  
演示时长：30~40 分钟  
角色：管理员 + 业务用户

## 1. 演示准备
1. 确认测试环境可访问，数据库迁移已完成。
2. 准备管理员账号与业务用户账号。
3. 打开以下页面：
  - `/tools/collection-workbench`
  - `/tools/paper-workspace`
  - `/documents/review`
  - `/chat`、`/agents`

## 2. 演示流程
1. 权限与会话策略
  - 展示管理员策略配置、角色权限边界。
  - 验证越权访问返回 403 并写入审计。
2. 专题采集工作台（WP-08/WP-10）
  - 启动 paper/patent 采集任务。
  - 演示暂停/继续/取消/重试控制。
  - 展示失败分类、批量入库、日志追踪、CSV 导出。
3. 论文工作台与查重闭环（WP-07）
  - 编辑并保存版本。
  - 提交查重、查看命中片段、导出报告。
4. 上传审核闭环（WP-05）
  - 上传文档、冲突处理、批量审核。
5. 安全策略与审计（WP-09）
  - 展示出网配置切换与审计查询。

## 3. 验收记录建议
1. 记录每一步成功/失败结果。
2. 记录对应 RTM 编号。
3. 记录发现问题与结论（通过/需整改）。
