# 管理员与业务用户培训材料（提纲）

更新时间：2026-03-14

## 1. 管理员培训（建议 90 分钟）
1. 系统架构与模块职责
2. 用户/权限组/会话策略配置
3. 任务中心运维
  - 任务监控指标
  - 异常任务恢复
  - 回滚与发布门禁
4. 数据安全策略
  - 内外网模式
  - 分级/脱敏/高敏拦截
  - 审计查询与取证
5. 故障排查与应急
  - 常见错误定位
  - 回滚流程演练

## 2. 业务用户培训（建议 60 分钟）
1. 上传、审核与知识库入库流程
2. 专题采集工作台
  - 启动任务
  - 查看状态与失败分类
  - 批量入库与结果追踪
3. 论文工作台与查重报告
4. 智能对话与检索实验台
5. 权限边界与合规注意事项

## 3. 培训产物建议
1. 录屏回放（管理员版/业务版）
2. 常见问题 FAQ
3. 演示账号与练习题
4. 培训签到与测验结果
