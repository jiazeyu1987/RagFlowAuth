# 知识库配置

## 全真链路状态

- 自动化状态：已接入 `doc/e2e` 全真链路
- 对应自动化：`fronted/e2e/tests/docs.knowledge-base-config.spec.js`
- 运行入口：`python scripts\run_doc_e2e.py --repo-root .`
- 真实数据来源：`scripts/bootstrap_doc_test_env.py` 提供真实受管目录、真实子管理员/审核员账号，以及文档 E2E 专用的预置空知识库 fixture
- 当前已覆盖：新建目录、重命名目录、新建知识库、保存知识库配置、刷新后仍存在、非空知识库删除保护、空知识库删除进入真实审批
- 当前未覆盖：返回上级、面包屑跳转、目录级删除、对话配置页签的自动化闭环

## 页签目标

用于管理知识库目录结构、创建或删除知识库，并维护知识配置与对话配置两类内容。

## 适用角色

- 子管理员

## 前置条件

- 当前账号已具备知识库配置访问权限。
- 系统中已存在至少一层目录和至少一个知识库，便于验证目录与知识库操作。
- 若要验证删除目录，应准备一个非根目录，可包含子目录，以便验证级联删除行为。
- 新建知识库后系统会自动生成 `ReadMe.txt`，因此新建库默认受“非空知识库”保护。
- 若要验证空知识库删除审批，应使用 bootstrap 预置的空知识库 fixture，而不是把刚新建的知识库当作空库。

## 按钮测试项

### 知识配置页签

- 所在区域：页内子页签切换区。
- 触发前提：当前停留在其他子页签。
- 点击模拟：点击“知识配置”。
- 预期结果：页面切换到知识库目录和知识库参数维护视图，可执行目录管理和知识库管理动作。

### 对话配置页签

- 所在区域：页内子页签切换区。
- 触发前提：当前停留在其他子页签。
- 点击模拟：点击“对话配置”。
- 预期结果：页面切换到对话相关配置视图，展示与聊天助手或对话行为相关的配置内容。

### 刷新

- 所在区域：知识库配置工具栏。
- 触发前提：当前位于任一知识库配置视图。
- 点击模拟：点击“刷新”。
- 预期结果：当前目录树、知识库列表和已保存配置重新加载为最新状态，便于管理员确认改动结果。

### 返回上级

- 所在区域：目录导航区域。
- 触发前提：当前不在目录根层级。
- 点击模拟：点击“返回上级”。
- 预期结果：页面回到当前目录的上一级，目录列表、面包屑和知识库内容同步切换到父级范围。

### 新建目录

- 所在区域：目录操作区。
- 触发前提：当前位于允许创建子目录的位置。
- 点击模拟：点击“新建目录”并完成目录名称填写。
- 预期结果：当前路径下新增一个目录节点，目录树和列表立即出现新目录，便于继续分层管理知识库。

### 重命名目录

- 所在区域：目录操作区。
- 触发前提：当前已选中一个允许重命名的目录。
- 点击模拟：点击“重命名目录”并完成新名称填写。
- 预期结果：该目录名称被更新，目录树、列表和面包屑中的路径文案同步变化。

### 删除目录

- 所在区域：目录操作区。
- 触发前提：当前已选中一个非根目录。
- 点击模拟：点击“删除目录”并在确认弹窗中确认删除。
- 预期结果：该目录及其子目录一并从当前结构中移除，页面自动切回父级目录，目录树和列表刷新后不再展示被删除内容。

### 新建知识库

- 所在区域：知识库操作区。
- 触发前提：当前位于允许创建知识库的目录。
- 点击模拟：点击“新建知识库”。
- 预期结果：系统打开“新建知识库”弹窗，展示名称、复制配置和挂载目录等字段，允许基于现有知识库配置快速创建。

### 面包屑路径按钮

- 所在区域：路径面包屑区域。
- 触发前提：当前处于多级目录路径中。
- 点击模拟：点击任意一个上级路径按钮。
- 预期结果：页面跳转到对应路径层级，目录内容和知识库列表同步切换到该位置。

### 保存

- 所在区域：已选中知识库的配置表单底部操作区。
- 触发前提：已选中一个知识库并修改了其配置内容。
- 点击模拟：点击“保存”。
- 预期结果：当前知识库配置被正式保存，刷新页面后仍保留最新配置，后续文档上传、检索或对话行为按新配置执行。

### 删除知识库

- 所在区域：已选中知识库的操作区。
- 触发前提：已选中一个当前允许删除的空知识库。
- 点击模拟：点击“删除知识库”并完成页面要求的删除流程。
- 预期结果：目标知识库从当前目录中移除，后续不再出现在知识库选择范围内，当前详情区同步清空或回到未选中状态。

### 关闭（新建知识库弹窗）

- 所在区域：新建知识库弹窗顶部操作区。
- 触发前提：新建知识库弹窗已打开。
- 点击模拟：点击关闭按钮。
- 预期结果：弹窗关闭且不创建新知识库，当前目录内容保持不变。

### 取消（新建知识库弹窗）

- 所在区域：新建知识库弹窗底部操作区。
- 触发前提：新建知识库弹窗已打开。
- 点击模拟：点击“取消”。
- 预期结果：弹窗关闭且本次录入内容不生效，不产生新的知识库记录。

### 创建（新建知识库弹窗）

- 所在区域：新建知识库弹窗底部操作区。
- 触发前提：已填写完整且合法的知识库名称，并确认复制配置来源与挂载目录。
- 点击模拟：点击“创建”。
- 预期结果：系统在所选挂载目录下创建新的知识库；若选择了复制来源，则新知识库继承对应配置，目录列表立即可见该知识库，并进入可继续配置状态。
